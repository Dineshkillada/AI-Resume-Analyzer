# 📄 AI Resume Analyzer

An AI-powered resume analyzer built with **Python + Streamlit + Claude API**.

## Features
- 📊 ATS score & job match score
- ✅ Matched & ❌ missing keywords
- 💪 Strengths & improvements
- 💡 Actionable tips to improve your resume

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy
Deployed on [Streamlit Cloud](https://share.streamlit.io)

## Tech Stack
- Python
- Streamlit
- Anthropic Claude API
- PyPDF2
